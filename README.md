# Web RTS Command Grid

An original browser-based real-time strategy prototype inspired by classic base-building RTS games.

The project must use original names, visuals, icons, maps, audio, and unit designs. It must not use Red Alert, Command & Conquer, or other protected game names, assets, UI art, unit identities, factions, logos, music, sounds, or story content.

## Target Prototype

- Resource harvesting and a visible resource economy
- Base construction with at least one production building
- Unit production, selection, movement, and attack commands
- Fog-free skirmish map suitable for quick browser testing
- Simple enemy AI that gathers, builds, and attacks
- Responsive web UI with clear controls and readable status panels

## Development

```bash
pnpm install
pnpm dev
pnpm build
pnpm test
```
